Settings

Your OpenAI API key is stored only in this browser (localStorage). It's used to call gpt-image-1 for the hardware / text cleanup, which produces noticeably cleaner results than the offline mode.

Per call cost ~$0.05–$0.19 at 'high' quality. About 4 calls per luggage (1 hardware + 3 outpaint borders).
